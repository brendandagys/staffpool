Purpose

Features


  • Storage and Security
    An encrypted database stores all submissions for future use.

  • Clarity
    View Code Red response data in real-time as it is entered, on the Code Red 'LIVE' tracker.

  • Communication
    A chat system in the Code Red response portal allows messaging between all staff involved.

  • Ease of Use
    Responsive Web design means that you can submit/view using your device of preference.
